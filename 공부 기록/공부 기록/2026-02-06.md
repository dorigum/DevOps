#### 전체 검색
![[Pasted image 20260206161423.png]]

| MainApp | 전체 검색 |
| ------- | ----- |
-> : `Service` param 정보 X
-> : `EndView` stArr 주소 전달

| Service | Student[] stArr = new Student[10] |
| ------- | --------------------------------- |
<- `MainApp` : Student[] 주소

| EndView | 배열을 순회하면서 출력 |
| ------- | ------------ |
-> `MainApp` stArr 주소 전달


#### 등록
![[Pasted image 20260206161819.png]]

| MainApp | 등록  |
| ------- | --- |
-> `Service` : 등록하려는 학생의 정보를 Student 객체로 만들어서 parameter 전달
-> `EndView` : 결과에 따른 메시지 전달

| Service |     |
| ------- | --- |
<- `MainApp` : 결과는 int 0(등록) / 1(중복 체크) / -1(배열 길이 초과)

| EndView | 전달받은 메시지 출력 |
| ------- | ----------- |
-> `MainApp` : 결과에 따른 메시지 전달
