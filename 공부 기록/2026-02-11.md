# Inheritance(상속)
#### 1. 정의
- 상속(Inheritance)이란, **기존 클래스(부모 클래스 또는 슈퍼클래스)의 속성과 동작(필드와 메서드)을 새로운 클래스(자식 클래스 또는 서브클래스)가 물려받는 기능**을 의미
- Java에서는 `extends` 키워드를 사용하여 상속을 구현

#### 2. 특징
- 자식 클래스는 부모 클래스의 **필드와 메소드를 상속받는다.**
- 자식 클래스는 부모 클래스의 기능을 **재사용**하거나 **오버라이딩(override)** 하여 기능을 변경할 수 있다.
- **단일 상속만 지원** (한 클래스는 하나의 부모 클래스만 상속 가능)
- 클래스 계층 구조를 형성하여 **계층적 구조 설계**가 가능

#### 3. 장점
- **코드 재사용성 향상**: 기존 코드를 다시 사용하여 중복을 줄임
- **유지 보수 용이**: 공통 기능을 부모 클래스에서 관리 가능
- **확장성 우수**: 새로운 기능을 자식 클래스에 추가하면서 기존 구조 유지 가능

#### 4. 단점
- **강한 결합도**: 부모 클래스 변경 시 자식 클래스에 영향이 큼
- **복잡한 계층 구조**로 인해 코드 이해 및 디버깅이 어려울 수 있음
- **다중 상속 불가**: 여러 클래스의 기능을 동시에 상속받을 수 없음 (단, 인터페이스로 보완 가능)

#### 5. 예제 코드
```java
// 부모 클래스
class Animal {
	void sound() {
		System.out.println("동물이 소리를 냅니다.");
	}
}

// 자식 클래스
class Dog extends Animal {
	@Override
	void sound() {
		System.out.println("강아지가 멍멍 짖습니다.");
	}
	
	void wagTail() {
		System.out.println("강아지가 꼬리를 흔듭니다.");
		}
	}
}
// 메인 클래스
public class InheritanceExample {
	public static void main(String[] args) {
		Dog dog = new Dog(); dog.sound(); // 오버라이딩된 메서드 실행
		dog.wagTail(); // 자식 클래스에만 존재하는 메서드
		}
}
```

class Parent {
	int a = 5;
	int b = 10;
}

	// 1번
	public void aa() {
		System.out.println();
	}
	
	// 2번
	public void bb() {
	}

class Child extends Parent {

	int b = 200;

	// 재정의

	public void aa() {
	}
	
	public void cc() {
	}

	// ㄱ
	Child ch = new Child();
	// ㄴ
	Parent p = new Child();
	
	// ㄴ의 경우
	if() {
	Child ch = (child)p;
	}
	
	// Parent p -----> Child
	/*
	Child ch ->
	a = 5, b = 10
	--------------------------
	1. aa()
	2. bb()
	--------------------------
	a = 100
	c = 200
	--------------------------
	3. aa()
	4. cc()
	--------------------------
	System.out.println(p.a) -> 5
	System.out.println(p.b) -> 10
	System.out.println(p.c) -> error
	
	p.aa(); -> 3번 ★★★★
	p.bb(); -> 2번
	p.cc(); -> error
	*/
